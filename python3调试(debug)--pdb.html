两种使用调试器的方法

可以这样启动调试：

  • python -m pdb filename.py
  • import pdb;pdb.set_trace()

下面的演示中使用的是第一种，实际中大多数会选择后者

从这个例子学到什么?

从这个例子我们可以学到:

  • 端点可以处可以设置命令
  • args可以显示传递给方法的参数
  • bt可以显示栈回溯
  • c表示继续直达断点
  • u和d可以回到上一层或者下一层栈
